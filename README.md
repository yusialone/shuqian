# Shuqian 书签站

一个基于 Next.js + Tailwind CSS + shadcn/ui 的现代书签静态站点。

## 本地开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
npm start
```

## 部署

推荐部署到 Cloudflare Pages、Vercel、Netlify 等平台。

## 目录结构

- `app/`：页面与布局
- `components/ui/`：UI 组件
- `styles/globals.css`：全局样式

## 特性
- 暗色模式
- 书签分组、搜索、添加、删除
- 动画与渐变背景
- 响应式设计
